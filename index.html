📤 엑셀 업로드

양식: 날짜 / 구분(수입·지출) / 내용 / 금액 열이 있는 엑셀 파일을 올리면 월별로 자동 정리됩니다. 같은 달 자료를 다시 올리면 해당 월이 새 내용으로 교체됩니다.

날짜구분내용금액삭제

잔액 = 이월금 + 수입 − 지출. 다음 달 이월금은 이번 달 잔액으로 자동 계산됩니다(직접 수정하면 그 값이 우선).

💰 월회비 납부 현황 월 5,000원

입금 내역의 내용에 단원 이름이 포함된 5,000원 단위 입금(건당 60,000원 이하)을 회비로 자동 집계합니다. 간식후원처럼 5,000원 단위가 아니거나 연회비를 넘는 입금은 제외되며, 잘못 집계된 건은 회비 내역에서 내용을 수정하면 반영됩니다.
